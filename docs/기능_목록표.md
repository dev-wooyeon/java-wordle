# 기능목록표

## MVP
- [x] **[domain] 오늘의 정답 단어를 하드코딩한다**
    - 예: `"APPLE"`
    - 게임 로직에 집중하기 위한 임시 처리

- [x] **[domain] Word 객체 구현 (길이만 검증)**
    - 문자열 래핑
    - 5글자가 아니면 예외
    - 알파벳 검증

- [x] **[test] Word 길이 검증 테스트**
    - 정상 케이스
    - 예외 케이스

- [x] **[domain] 문자 판정 결과 enum 추가**
    - GREEN
    - YELLOW/GRAY

- [x] **[domain] 입력 단어와 정답 비교 로직 구현**
    - 같은 위치 + 같은 문자 → GREEN
    - 그 외 → YELLOW/GRAY

- [x] **[test] 기본 판정 로직 테스트**
    - 완전 일치
    - 완전 불일치

- [x] **[domain] Game 클래스 구현**
    - 정답 보관
    - 시도 횟수 관리
    - 종료 여부 상태

- [x] **[domain] 게임은 최대 6번 입력받고 종료된다**
    - 정답을 맞히면 즉시 종료
    - 6번 시도 후 종료

- [x] **[test] Game 종료 조건 테스트**
    - 6턴 초과 불가
    - 정답 성공 시 종료

- [ ] **[ui] 콘솔 입력으로 단어를 입력받는다**
- [ ] **[ui] 판정 결과를 콘솔에 출력한다**
    - GREEN / NOT_GREEN 문자열 출력

- [ ] **[app] main에서 Game과 UI를 연결한다**
    - while 루프 기반
    - `System.exit()` 사용 ❌
    - 자연 종료
